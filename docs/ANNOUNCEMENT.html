Knowledge Infrastructure

Wiki-Weaver

Compiles a pile of raw sources into a connected, error-free, provenance-tracked memory — instead of stateless RAG or one-shot summaries that rot.

What It Is

A declarative convergence loop.

Andrej Karpathy's "LLM Wiki / second brain," made real: compile sources into one-concept-per-page notes that cross-link and merge. Authored as a graph — nodes and edges in DOT, run by the attractor engine — not as imperative script.

The shape

Steps, not statements

Five nodes wired by edges. Each is an isolated session with a thin, scoped context. It loops until the wiki is structurally valid and quality-accepted.

The split

Code controls, agents judge

Code owns control, validation, archiving, and dedup. Agents do only the irreducible judgment — synthesis, quality, remediation.

Why A Pipeline

Specialists with quality gates — not one big prompt in a loop.

The honest baseline is the real medium-tools CLI — a useful acquisition tool (login → sync → download → summarize) that fills the _inbox/ with one-shot per-article summaries. Acquiring is a different job from compiling a connected wiki. That gap is where the loop earns its place.
